Setup checks

This page checks if the app setup is correct. All items on this list should be green:

  1. CSS must be properly handled (static files)
  2. All requests should be forwarded from Nginx
  3. PostgreSQL database must be reachable
  4. Redis must be reachable
  5. There must be at least 1 running instance of Celery
  6. Kafka should be up and able to send messages to a topic
